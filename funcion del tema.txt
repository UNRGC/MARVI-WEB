Para que el tema funcione en todas las paginas hay que importar la hoja de estilos variables.css a la hoja de estilos nueva

Importar pegando esta etiqueta al inicio de la hoja de estilos:
@import url(variables.css); /* Importa las variables de colores */

Dentro de la hoja de estilos variables.css encontraras los colores a usar

Ejemplo de uso:
button {
    background-color: var(--color-primario)
    color: var(--texto primario)
}

Para que los temas se apliquen en autom√°tico hay que importar el script temas.js al inicio del html

importar pegando la etiqueta al inicio del html:
<!-- Script del tema -->
<script src="../js/temas.js"></script>

ejemplo de html:
<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>MARVI</title>
    <!-- Estilos de bootstrap -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous" />
    <!-- Iconos de bootstrap -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css" />
    <!-- Script del tema -->
    <script src="../js/temas.js"></script>
</head>

Puntos importantes:
* si no se importa la hoja de estilos variables.css en la hoja de estilos que se usara no funcionaran los colores
* Si no se usan los colores variables el tema no funcionara en esa pagina
* El script temas.js se debe importar siempre abajo de la hoja de estilos que se usara
